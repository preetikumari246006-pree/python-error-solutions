# GitHub Email Verification Issue 📧

## ❓ What Is This Problem?

Sometimes GitHub asks you to verify your email address
before you can push code or use certain features.

---

## 🤔 Why This Happens

- Email not verified after signup
- Wrong email added to account
- Email confirmation link expired

---

## 🛠 Step-by-Step Fix

### ✅ Step 1: Check Verification Email

1. Open your email inbox
2. Search for: GitHub
3. Open the verification email
4. Click **Verify email address**

---

### ✅ Step 2: Resend Verification Email

1. Go to GitHub
2. Click your profile picture
3. Go to **Settings**
4. Click **Emails**
5. Click **Resend verification email**

---

### ✅ Step 3: Check Spam Folder

Sometimes the verification email goes to spam.

---

## 💡 My Experience

When I created my GitHub account,
I could not push code because my email was not verified.

After verifying the email,
everything worked normally.

---

## 🎯 Tip

Always verify your email immediately after creating a GitHub account.
